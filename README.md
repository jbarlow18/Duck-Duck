# Duck-Duck
A platformer game where the objective is to dodge spikes and collect magical cheese to help you maneuver the level. The Magical cheese increases your quantity of jumps which can be used strategically.

<h2>Controls</h2>
<p>w/SPACE = Jump<br>
a/LeftNum = Left<br>
d/RightNum = Right<br>
s/Down = Crouch<br>
LShift = Glide</p>
